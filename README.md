
# Upload audio and make timestamp note



## Run Locally

Clone the project

```bash
  git clone https://github.com/Sumit45doc/saarthi-assignment.git
```

Go to the project directory

```bash
  cd saarthi-assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Deployment

To deploy this project run

```bash
  npm run build
```

```bash
  serve -s build
```
